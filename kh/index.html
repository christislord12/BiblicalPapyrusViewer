Ketef Hinnom Silver Scrolls

The Ketef Hinnom Silver Scrolls are an archeological artifact that dates back to the First Temple Period and contain the earliest inscription containing text from the Hebrew Bible. This is an informational website created by some archeology fanatic who is really interested in the Ketef Hinnom Silver Scrolls and has absolutely no formal archeological or theological education!
